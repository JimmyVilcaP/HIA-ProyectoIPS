Health Insights Agent (HIA)

Aplicación basada en Streamlit que permite analizar reportes médicos en PDF utilizando Inteligencia Artificial.

📦 Repositorio GitHub 🚀 Despliegue Streamlit Cloud

Presentación de Hito 2

Este sitio sirve como repositorio centralizado de documentación y evidencias de ingeniería para HIA. Actualmente el proyecto ha completado de forma satisfactoria los Sprints 0 y 1, encontrándose en fase de desarrollo y validación final para el cierre del Sprint 2. El sistema base se encuentra funcional de manera local y desplegado en entornos en la nube con su suite de integración automatizada activa.

Ecosistema Tecnológico Implementado

Frontend / Interfaz

Python y Streamlit (Auth Pages, Sidebar, Analysis Form, Chat Interface).

Procesamiento e IA

Groq API, LangChain, FAISS, HuggingFace Embeddings y PDF Extractor.

Datos y Cloud

Supabase (PostgreSQL) para la persistencia relacional de datos y Streamlit Cloud para distribución.
