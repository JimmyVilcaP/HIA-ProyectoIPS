Cronograma Real del Proyecto

Sprint Objetivo Actividades Realizadas / Planificadas Estado
Sprint 0 Planificación inicial del proyecto. Selección del proyecto Open Source HIA, Análisis de arquitectura, Creación del repositorio, Configuración de GitHub Projects, Elaboración del cronograma y Definición de roles. Completado
Sprint 1 Implementar la infraestructura DevOps inicial. Configuración de GitHub Actions, Creación de ramas main y develop, Integración de pruebas automatizadas, Organización del backlog y Configuración de Jira. Completado
Sprint 2 Publicar y validar el sistema. Despliegue en Streamlit Cloud, Publicación de GitHub Pages, Validación de integración con Supabase, Validación de integración con Groq y Evidencias funcionales. En desarrollo / Completado
Sprint 3 Implementar mejoras funcionales y optimización. Mejoras de experiencia de usuario, Mejoras en interacción con IA, Optimización visual y Pruebas complementarias. Pendiente
Sprint 4 Documentación y cierre del proyecto. Elaboración del informe IEEE, Consolidación de evidencias, Documentación técnica y Presentación final. Pendiente
