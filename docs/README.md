# Health Insights Agent (HIA) - Web de Documentación Académica

Este subdirectorio contiene la infraestructura estática completa que sirve de base para el despliegue del proyecto universitario HIA en **GitHub Pages**.

## 📁 Guía de Recursos Visuales

Para mantener el diseño limpio y profesional, sustituye los marcadores visuales del HTML añadiendo tus archivos en las siguientes rutas específicas dentro de `docs/`:

### Carpeta `docs/assets/diagrams/`
* `architecture_map.png`: Diagrama de componentes multicapa (Capa de Presentación, Negocio, Datos).
* `cicd_pipeline.png`: Esquema visual de ejecución de tus automatizaciones en GitHub Actions.

### Carpeta `docs/assets/images/`
* `evidencia_github_actions.png`: Captura de pantalla de tus workflows en verde.
* `evidencia_supabase.png`: Captura del panel de tablas relacionales de PostgreSQL en Supabase.
* `evidencia_login.png`: Captura de pantalla de la interfaz de login nativa de la app en Streamlit.
* `evidencia_pdf_analysis.png`: Captura del procesamiento de reportes médicos.
* `evidencia_chat_ia.png`: Captura del agente interactuando y respondiendo consultas basadas en el PDF.
* `evidencia_streamlit_cloud.png`: Estado del despliegue en los servidores de Streamlit Cloud.
